# 支持鸿蒙平台的OneNET MQTT接入协议

在鸿蒙系统上实现OneNET 平台接入，基于MQTT协议。
使用前请先下载 鸿蒙系统的 MQTT 库：https://gitee.com/qidiyun/harmony_mqtt
